# DEC/HEX converter

Simple DEC2HEX and HEX2DEC converter. Just to try out Rust combined with 
ui package Slint.


## Slint Rust Template

The app is build on the Slint Rust template, as available on the website of Slint. 

## Usage


1. Build with `cargo`:
    ```
    cargo build
    ```
2. Run the application binary:
    ```
    cargo run
    ```
